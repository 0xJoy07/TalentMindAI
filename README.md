TalentMind AI – Workforce Intelligence & Career Readiness Agent

Overview

TalentMind AI is an AI-powered workforce intelligence platform that helps organizations identify the best candidates, recommend personalized learning paths, assess readiness levels, and provide workforce analytics.

The project combines recruitment intelligence and employee development into a single AI-driven solution.

---

Problem Statement

Recruiters often rely on keyword matching, which can miss highly suitable candidates. Organizations also struggle to identify skill gaps and recommend appropriate learning resources.

TalentMind AI addresses these challenges through intelligent candidate ranking, learning recommendations, readiness assessment, and workforce analytics.

---

Key Features

Candidate Ranking Agent

- Matches candidates against job descriptions
- Calculates suitability scores
- Provides ranked candidate recommendations

Learning Path Agent

- Generates role-based certification recommendations
- Suggests personalized learning plans
- Supports career growth and upskilling

Assessment Agent

- Evaluates candidate readiness
- Uses score and study-hour metrics
- Provides improvement recommendations

Analytics Dashboard

- Displays workforce readiness metrics
- Highlights top performers
- Identifies at-risk employees

Installation

1. Clone the repository

git clone <repository-url>

2. Navigate to the project directory

cd TalentMindAI

3. Install dependencies

pip install -r requirements.txt

4. Run the application

python app.py

5. Open in browser

http://127.0.0.1:5000
